# PdfOperations

A console application for working with PDF files, images, and office documents.

## Features

- Convert PDF files to TXT
- Convert PDF files to images
- Convert images to PDF
- Run OCR on images and save the result as TXT
- Merge PDF files
- Split PDF files
- Extract PDF information
- Diagnose PDF fonts
- Extract embedded images from PDF files
- Extract PDF attachments

## Requirements

The application uses external tools:

- LibreOffice
- Poppler
- ImageMagick
- Tesseract OCR
- QPDF
